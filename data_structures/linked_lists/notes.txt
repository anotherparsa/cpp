A linked list is a data structure that consists of a sequence of elements, each of which is called a node.
there are:
Singly Linked List, Singly Circular Linked List, Doubly Linked List and Doubly Circular Linked List.
each node consists of two main part:
Data: 
the value that the node holds.
Pointer:
This is a pointer to the next or previous node in the sequence. It allows for the linking of nodes to form the list.
Unlike arrays:
they're not fixed size, they can grow or shrink.
elements doesn't have to be homogeneous and can be heterogeneous.
elements aren't stored continuously and are stored in different places in the memory.
A linked list can have three properties:
Head node pointer:
A node pointer which points to the first element of the list.
Tail node pointer:
A node pointer which points to the last element of the list.
Length variable:
keepts track of the number of total elements in the list.

Singly Linked List:
EX:
empty Singly linked list:
                                                      head = null
                                                      tail = null
                                                      length = 0 
                                                      
linked list with some elements:                       
                                                      head = 1
(1) -> (2) -> (3) -> (4) -> null                      tail = 4
                                                      length = 4

Appending a Node(O(1)):
When adding a node to the end of a Linked list, there two cases, either the list is empty or it has at least one node already.
if the link list is empty:
1-Create a new_node.
                                                      head = null
(5) -> null                                           tail = null    new_node = 5
                                                      length = 0 

2-both head and tail are going to point to the new_node since there is only one element.
                                                      head = 5
(5) -> null                                           tail = 5
                                                      length = 0

3-the length increases by one.
                                                      head = 5
(5) -> null                                           tail = 5
                                                      length = 1

if the list has at least one element:
1-Create a new_node.
                                                      head = 1
(1) -> (2) -> (3) -> (4) -> null     (5) -> null      tail = 4    new_node = 5
                                                      length = 4

2-Current tail node's(the last node) next will be new_node and links the new_node to the list.
                                                      head = 1
(1) -> (2) -> (3) -> (4) -> (5) -> null               tail = 4    new_node = 5
                                                      length = 4

3-the tail pointer will point to the new_node and makes it the new tail node.
                                                      head = 1
(1) -> (2) -> (3) -> (4) -> (5) -> null               tail = 5    new_node = 5
                                                      length = 4      

3-the length increases by one.
                                                      head = 1
(1) -> (2) -> (3) -> (4) -> (5) -> null               tail = 5    new_node = 5
                                                      length = 5      

*if we didn't have a tail node pointer, we would have to traverse the entire linked list till the last node starting from the head node pointer
and then add the new node to the list. which is going to be O(n) but since we already have a pointer to the last element it's O(1)

Deleting the last Node(O(N)):
when deleting the last element of a linked list, there are three cases, the list is empty, the list only has one element, the list has more than one element.
if the link list is empty:
1-just break out of the function
                                                      head = null
                                                      tail = null
                                                      length = 0 

if the list has one element:
1-since there is only one element, both head and tail node pointer are pointing to it, they both have to point to null pointer.
create a temp_node which is going to be deleted.
                                                      head = null  
(1) -> null                                           tail = null    temp_node = 1    
                                                      length = 1

2-decrease the length
                                                      head = null  
(1) -> null                                           tail = null    temp_node = 1    
                                                      length = 0

3-delete the temp node:
                                                      head = null
                                                      tail = null
                                                      length = 0 

if the list has more than one element:
1-get to the previous node of the current tail node pointer, becuase that is going to be the new tail node.
create a temp_node which is going to be deleted.
                                                      head = 1    previous = 4
(1) -> (2) -> (3) -> (4) -> (5) -> null               tail = 5    temp_node = 5
                                                      length = 5

2-the previous node will become the new tail node.
                                                      head = 1    previous = 4
(1) -> (2) -> (3) -> (4) -> (5) -> null               tail = 4    temp_node = 5
                                                      length = 5

3-the new tail pointer will point to null and unlinks the last node from the list.
                                                      head = 1    previous = 4
(1) -> (2) -> (3) -> (4) -> null (5) -> null          tail = 4    temp_node = 5
                                                      length = 5

4-decrease the length
                                                      head = 1    previous = 4
(1) -> (2) -> (3) -> (4) -> null (5) -> null          tail = 4    temp_node = 5
                                                      length = 4
5-delete the temp node.
                                                      head = 1    previous = 4
(1) -> (2) -> (3) -> (4) -> null                      tail = 4    
                                                      length = 4

insert node(o(N)):
when inserting node to a linked list, there are three cases, you're inserting at the first index, you're inserting at the last index, you're inserting somewhere between.
inserting at the first index or index 0:
it's just like prepend.
inserting at the last index or index length - 1:
it's just like append.
inserting somewhere between (inserting at index 1):
1-Create a new node.
                                                      head = 1
(1) -> (2) -> (3) -> (4) -> null                      tail = 4   new_node = 5
             (5) -> null                              length = 4

2-get the previous node of the index you want to insert.
                                                      head = 1   previous_node = 2
(1) -> (2) -> (3) -> (4) -> null                      tail = 4   new_node = 5
             (5) -> null                              length = 4

3-new node will point to the previous_node's next node.
                                                      head = 1   previous_node = 2
(1) -> (2) -> (3) -> (4) -> null                      tail = 4   new_node = 5
             (5) -> (4)                              length = 4

4-previous_node will point to the new node and linking is complete.
                                                      head = 1   previous_node = 2
(1) -> (2) -> (3) -> (5) -> (4) -> null               tail = 4   new_node = 5
                                                      length = 4   

5-increase the length.
                                                      head = 1   previous_node = 2
(1) -> (2) -> (3) -> (5) -> (4) -> null               tail = 4   new_node = 5
                                                      length = 5

deleting the last index or index length - 1:

reversing a linked list:
we're going to change the direction of the nodes pointing to each other.
1-swap the head and tail node.
                                                      head = 5    
(1) -> (2) -> (3) -> (4) -> (5) -> null               tail = 1    
                                                      length = 5  

2- we need three variables, before, current and after node pointer to keep track as we change.
1-swap the head and tail node.
                                                      head = 5    
null <- (1) <- (2) <- (3) <- (4) <- (5)               tail = 1    
                                                      length = 5

doubly linked list:
unlike singly linked list, nodes in the doubly linked list are going to have both next and previous node pointers.
they are linked in both ways.
EX:
empty doubly linked list:
                                                      head = null
                                                      tail = null
                                                      length = 0 
                                                      
linked list with some elements:                       
                                                      head = 1
null <-> (1) <-> (2) <-> (3) <-> (4) <-> null         tail = 4
                                                      length = 4
